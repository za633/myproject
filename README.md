# Thebes Game Hub

A responsive web-based gaming website built using HTML, CSS, and JavaScript.

## Overview
This project is a simple frontend website that showcases game-related content with a clean and responsive layout.

## Features
- Responsive design for different screen sizes
- Organized sections for content display
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript
